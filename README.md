# ЦПЦ Обука — Категорија Ц (превоз терета)

Веб апликација за припрему за ЦПЦ испит, категорија Ц.

## 📊 Шта је унутра
- **607 питања** у 6 тематских области
- **14 питања са сликама** (све слике су већ у пројекту - не треба GitHub)
- **3 режима**: вежба, симулација испита (30 пит, 75%), понављање грешака
- **Админ панел** са комплетним мониторингом

## 🔒 Заштита (СВЕ ИНТЕРНО)

Слике се не хостују на GitHub-у. Налазе се у пројекту и сервирају се **само пријављеним полазницима**:
- Директан приступ `/protected-images/` је блокиран
- Слике иду кроз `/img/наслов.png` који захтева ауторизован JWT cookie
- Cache-Control: no-store (свака слика се проверава)
- Referer check (спречава hotlinking)
- X-Frame-Options: DENY (спречава iframe embed)
- Сваки приступ слици се логује (`[IMG] email → filename`) - можеш видети ко скида слике

Plus полазнички интерфејс:
- Watermark преко екрана са имејлом + временом
- Lock на један уређај (browser+OS fingerprint)
- Детекција DevTools / F12 / PrintScreen / Ctrl+P
- Tab blur protection
- HttpOnly cookies + JWT 30 дана

## 🚀 Покретање на Replit

### Корак 1: Импорт
1. **replit.com** → ⊕ Create → **Import code or design** → **Zip file**
2. Учитај **cpc-kurs-c.zip**
3. Назив: `cpc-kurs-c`
4. Сачекај да Replit инсталира зависности (~1 минут)

### Корак 2: Secrets (🔒 у левом менију)
Додај ова два:

| Кључ | Вредност |
|---|---|
| `ADMIN_PASSWORD` | `ADMIN381` (или своја јака лозинка) |
| `JWT_SECRET` | насумичан низ од 32+ знака |

Пример JWT_SECRET (генериши свој!): `cpc2026xK3p9Lm5Qq7v2bN4hT8jR1aZ6wY0`

### Корак 3: ▶ Run
Конзола треба да покаже:
```
🚛  CPC Obuka — Kategorija C
🌐  Server: http://localhost:3000
👤  Admin:  http://localhost:3000/admin
📊  Pitanja u bazi: 607
```

### Корак 4: Publish
1. Кликни **Publish** горе десно
2. Изабери **Reserved VM** (пошто имаш плаћен налог - увек активан)
3. Назив: `cpc-kurs-c` → твоја адреса: `cpc-kurs-c.replit.app`

## 🛠 Чести задаци

### Ресет уређаја (полазник променио телефон)
Админ → пронађи полазника → **Ресет уређаја**. Може поново да се пријави на новом.

### Промена лозинке полазнику
Админ → пронађи полазника → **Нова лозинка** → саопшти му нову.

### Месечна промена админ лозинке
Replit → Secrets → измени `ADMIN_PASSWORD` → Republish.

### Праћење грешака
- **По полазнику**: у админ листи свакоме пише топ 5 грешака
- **Глобално**: секција "Најтежа питања" - видиш која питања сви најчешће промашују
- **CSV извоз**: ↓ CSV дугме - све у Excel

### Backup
- База полазника: `data/cpc.db` (можеш је преузети из Replit фајл-експлорера)
- Још једноставније: CSV из админ панела

## ❓ Проблеми

| Проблем | Решење |
|---|---|
| Сервер не стартује | Постављен ли је `JWT_SECRET` у Secrets? |
| Слике се не приказују | Проверити фолдер `public/protected-images/` - треба да има 14 PNG-ова |
| "Налог се користи на другом уређају" | Админ → Ресет уређаја |
| `better-sqlite3` греша при `npm install` | Replit има `nodejs-20` модул који има све build алате. Ако ипак: упиши у Shell `npm rebuild better-sqlite3` |
